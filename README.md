# React Learning Projects

This repository contains a collection of React.js projects created for learning and practicing various concepts in React development. Each project demonstrates different features, hooks, and integrations commonly used in modern React applications.

## Projects Included

### 1. APIcalling
A project demonstrating how to make API calls in React using fetch or axios.

### 2. Child2Parent
Examples of passing data from child components to parent components in React.

### 3. ContextAPI
Implementation of React's Context API for state management across components.

### 4. FormHandling
Various form handling techniques in React, including controlled and uncontrolled components.

### 5. Function
Functional components and their usage in React applications.

### 6. GalleryProject
A photo gallery application built with React.

### 7. LocalStorage
Using browser's localStorage for data persistence in React apps.

### 8. NotesApp
A simple notes application with CRUD operations.

### 9. React-IntershipCard
An internship card component project.

### 10. React-WorkCard
A work card component project.

### 11. ReactRouter
Basic routing implementation using React Router.

### 12. Router-Advanced
Advanced routing techniques and nested routes in React.

### 13. Tailwind css
Integration of Tailwind CSS with React for styling.

### 14. TwoWayBinding
Two-way data binding examples in React.

### 15. UseEffect
Demonstrations of the useEffect hook for side effects.

### 16. useState
Basic usage of the useState hook for state management.

## Getting Started

Each project is self-contained and can be run independently. To run a specific project:

1. Navigate to the project directory:
   ```bash
   cd <project-name>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173` (or the port specified by Vite).
