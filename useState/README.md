  useEffect-handles side effects like data fetching, subscriptions, or manual DOM manipulations
  useMemo-memoizes expensive calculations to optimize performance
  useCallback-memoizes functions to prevent unnecessary re-renders of child components
  useLayoutEffect-similar to useEffect but fires synchronously after all DOM mutations
  useImperativeHandle-customizes the instance value that is exposed when using ref in parent components
